# Chrome-auto-refresher
Chrome extension for auto-refreshing error pages
네트워크 오류 및 대기열 자동 새로고침 크롬 확장프로그램 

-----

### 📝 설명문

특정 사이트(이벤트 페이지 등) 접속 시 발생하는 서버 오류(502, 503 등) 및 접속 대기열 화면을 감지하여 자동으로 새로고침해 주는 확장프로그램입니다.
새로고침 주기(기본 2초)를 임의로 너무 짧게 줄일 경우, 대상 서버의 트래픽 방어 정책에 위반되어 IP 차단(Ban)이 발생하는 경우가 생길 수 있습니다.  
그러니 가급적 기본 시간을 유지하는 것을 권장합니다.

### ⚙️ 설치 및 사용 방법
1.  저장소의 코드를 다운로드(`Code` -\> `Download ZIP`)하여 압축을 해제합니다.
2.  크롬 브라우저 주소창에 `chrome://extensions/`를 입력하여 접속합니다.
3.  우측 상단의 \*\*'개발자 모드'\*\*를 켭니다.
4.  좌측 상단의 \*\*'압축해제된 확장 프로그램을 로드합니다'\*\*를 클릭하여 압축을 해제한 폴더를 선택하면 설치가 완료됩니다.
5.  확장프로그램 아이콘을 클릭하여 타겟 URL을 입력 후 시작 버튼을 누르시면 됩니다.

### 📌 참고 사항
